Inicia el proyecto
Ejecuta el siguiente comando en el directorio raíz de tu app:
firebase init
Especifica tu sitio en firebase.json
Agrega el ID de tu sitio al archivo de configuración firebase.json. Después de 
realizar la configuración, consulta las prácticas
 recomendadas para implementar varios sitios.
{

  "hosting": {

    "site": "seungs-solution",

    "public": "public",

    ...

  }

}
Cuando estés listo, implementa tu app web

Ubica los archivos estáticos (p. ej., HTML, CSS y JS) en 
el directorio de implementación de tu app 
(el directorio predeterminado es “public”). Luego, ejecuta este 
comando desde el directorio raíz de la app:
agregar yarn build para gestionar la web en react
firebase deploy --only hosting:seungs-solution
